Cooperative Online Trajectory Planning for Autonomous Aerial Robotics Swarm

Nov 11, 2022

News

  • November 11, 2022. That’s it, this project is started!

Project objectives

This project aims to address the issue: How to enable agile flight for aerial robotics swarm in complex environments? The main considerations of this projects include:

  • Full autonomy: Fully onboard perception, localization, planning, and control.
  • Online: Real-time flying in unknown environments.
  • Cooperation: Multiple drones to perform cooperative tasks, such as search and rescue.

Roadmap

  • WP1: Trajectory parameterization

    Parameterize trajectories in a spatial-temporal-decoupled form to support independent optimization on the spatial and temporal profile of the trajectory.

  • WP2: Distributed trajectory planning

    Build an optimization framework considering control effort, execution time, dynamic feasibility, obstacle avoidance, and swarm reciprocal avoidance. And use gradient-based solvers to solve the problem.

  • WP3: Real-world experiments

    Build quadrotors equipped with depth cameras, Intel NUC, and the Pixhawk controller to conduct real-world experiments.
